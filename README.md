# 📊 Exploratory Data Analysis & Business Insights

## 📌 Overview

This project focuses on exploring a retail dataset to uncover patterns, trends, and relationships that can support business decision-making. The analysis emphasizes understanding data distributions, identifying key drivers of performance, and generating actionable insights.

---

## 📂 Dataset Description

The analysis is based on a retail dataset containing information related to:

* Sales transactions
* Product categories
* Customer segments
* Order and time-based details

The dataset was preprocessed and cleaned prior to analysis.

---

## 🔍 Analysis Workflow

### 🔹 Data Understanding

* Reviewed dataset structure and key variables
* Identified relevant fields for business analysis

### 🔹 Exploratory Data Analysis

* Generated summary statistics to understand distributions
* Analyzed sales performance across categories and segments
* Examined time-based trends to identify patterns

### 🔹 Relationship Analysis

* Explored relationships between variables using correlation techniques
* Identified factors that may influence sales and performance

---

## 📈 Visualizations

The following visualizations were created to support insights:

* **Sales Distribution Plot** → to analyze variability in sales
* **Monthly Sales Trend** → to observe changes over time
* **Category-wise Analysis** → to compare performance across categories
* **Correlation Heatmap** → to identify relationships between variables

All visual outputs are available in the `visuals/` directory.
