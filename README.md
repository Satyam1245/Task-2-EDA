# Task 2: Exploratory Data Analysis & Business Intelligence

## 📌 Objective

The objective of this task was to explore the dataset in detail and extract meaningful insights that can support business decisions. This involved analyzing data distributions, identifying trends, and answering key business-related questions.

---

## 📂 Dataset Overview

The analysis is based on a retail dataset (Superstore), which contains information related to:

* Sales transactions
* Product categories
* Customer segments
* Time-based order details

The dataset was previously cleaned and prepared for analysis.

---

## 🔍 Analysis Workflow

### 1. Data Understanding

* Reviewed dataset structure and variables
* Identified key fields relevant for analysis

### 2. Exploratory Data Analysis

* Generated summary statistics to understand data distribution
* Examined sales performance across categories and segments
* Analyzed time-based trends to observe changes over months

### 3. Relationship Analysis

* Explored relationships between variables using correlation
* Identified factors influencing sales performance

---

## 📈 Visualizations

The following visualizations were created to support the analysis:

* **Sales Distribution Plot** → to understand variability in sales
* **Monthly Sales Trend** → to identify patterns over time
* **Category-wise Analysis** → to compare performance across categories
* **Correlation Heatmap** → to examine relationships between variables

All visual outputs are available in the `visuals/` folder.
